Github

Git merupakan software yg berbasis Version Control System ( VCS ) yang memiliki tugas untuk mencatat perubahan seluruh file atau repository suatu proyek, Biasanya Developer software menggunakan Git untuk distributed revision, hal ini dikarnakan penyimpanan database Git tidak hanya berada di satu tempat. Git terdapat merge untuk menyebut aktifitasi penggabungan kode, sedangkan pada VCS yang terpusat database disimpan disimpan di satu tempat dan setiap perubahannya disimpan ke sana. Git pertama kali dikenal pada tahun 2005 dan semakin disini Git semakin populer dan sekarang menjadi DVCs yang paling populer didunia. Git juga termasuk tool yang terpopuler sehingga banyak perusahaan yang terkenal menggunakan Git Git tidak membutuhkan server untuk melakukan perubahan atau melihat riwayat revisi, hal ini dikarnakan dalam Git, semua riwayat proyek akan selalu di duplikasi, baik itu diserver ataupun di local computer
